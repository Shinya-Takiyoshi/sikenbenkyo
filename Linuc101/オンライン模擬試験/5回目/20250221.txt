■プログラム「file」をどのユーザが実行しても、プログラムの所有者の権限で実行させたい。なお、「file」のパーミッションは755である。
chmod 4755 file
chmod u+s file
プログラムをどのユーザが実行しても、プログラムの所有ユーザ権限で実行されるようにするにはSUIDを設定します。SUIDは特殊なパーミッションの1つです。
4000を超える値にするか、u+s(userにsという権限を追加、sはおそらくsudoかsuper userのこと)

■prコマンドで「httpd.conf」ファイルを整形する際、ヘッダに表示されるファイル名を「testfile」に変更したい。また、1ページあたり30行としたい
pr -h testfile -l 30 httpd.conf
lは行数 lineのこと
指定行数は数字のみ。
■「file」ファイルを500行ごとに分割したものを別のファイルに保存したい。保存するファイル名は決めない。
split -500 file hoge
適当にxaa xabとファイル名が設定される。

■「httpd.conf」ファイルの内容を1行あたり30文字に整形して表示したい。
fmt -w 30 httpd.conf
-wはword文字数ということ。

■「file1」と「file2」ファイルを行単位で区切り文字「:」を使用して結合したい。
joinとの違いは？
join:共通するフィールドがある行を結合して表示する
paste:複数のファイルを行単位で結合する
paste -d : file1 file2
つまり、joinはSQLのjoinのようなもの。共通項が必要。
それに対してpasteは無条件に結合する。
